# meigen-plugin

WordPressで名言を管理するためのプラグインです。

## 言語設定

**【重要】日本語での入出力を必須とする - 英語での応答は厳禁**
- **すべての応答は必ず日本語で行うこと**（英語での応答は一切禁止）
- このプロジェクトの開発チームは日本語話者のため、Claude Codeとのやり取りは必ず日本語で行う
- コードコメント、コミットメッセージ、ドキュメント作成も日本語で統一する
- エラーメッセージや説明も日本語で提供する
- **英語で質問されても必ず日本語で応答すること**

## 名言投稿画面で本を同時登録する

名言の編集画面に「新規の本を同時登録する」フォームを追加しました。タイトルを入力すると保存時に本（books投稿タイプ）が自動作成され、購入リンクや著者も合わせて保存されます。作成された本はそのまま出典に紐づきます。既存の本を出典にしたい場合は従来通りセレクトボックスから選択してください。
